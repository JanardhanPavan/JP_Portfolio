## 📖 About

This is the source code for my personal portfolio site, built to present my background, skills, and selected projects in one place for recruiters, collaborators, and visitors.

## ✨ Features

- Responsive design that works across desktop, tablet, and mobile
- Sections for About, Skills, Projects, and Contact
- Clean, modern UI built from a custom Figma design
- Fast load times with a component-based architecture

## 🛠️ Built With

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Bootstrap](https://getbootstrap.com/)
- [Figma](https://figma.com/) (design)

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/JanardhanPavan/JP_Portfolio.git

# Move into the project directory
cd JP_Portfolio

# Install dependencies
npm install

# Start the development server
npm start
\`\`\`

The app will run at \`http://localhost:3000\`.

### Build for Production

\`\`\`bash
npm run build
\`\`\`
